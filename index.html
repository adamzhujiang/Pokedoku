How to Play!

  • You have 9 total guesses (PP: 9/9). Use them wisely! You can only win if you guess 9/9 Correctly!
  • Click any square on the grid to enter a Pokémon name.
  • Currently only gen 1-3 pokemon are supported
  • The Pokémon must match both the row and column category for that square.
  • Each Pokémon can only be used once per game.
  • If the entry is correct, it will fill the square and add to your score.
  • If the entry is incorrect, it will cost 1 PP but won't fill the square.
  • For the Mega Pokémon category, use the base form (ex: Gengar instead of Mega Gengar)
  • In addition, for the Mega Pokémon category, use the typing of the pre-Mega Pokémon (ex: Charizard would not work for Dragon)
  • The game ends when you run out of PP or fill all 9 squares.
  • Use the difficulty buttons (Poké Balls) to change the challenge level.
  • Click "Give Up" to reset the board and try again!
